## 简介

> nrm 学习笔记。

## 镜像源管理

```text
安装
npm i nrm -g

添加
nrm add qingting http://ip:4873

查看
nrm ls

切换
nrm use qingting
```

## 镜像配置

```text
查看
npm config ls

设置
npm config set <key> <value> [-g|--global]

获取
npm config get <key>

删除
npm config delete <key>

修改
npm config edit
```
